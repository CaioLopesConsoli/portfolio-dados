# Projetos Python — Engenharia de Dados

Scripts desenvolvidos para simular tarefas reais de engenharia de dados usando Python puro.

## Projetos

### 01 — Pipeline de Análise de Vendas
Processa registros de vendas do primeiro trimestre e gera um relatório com faturamento total, por categoria, por região, por vendedor e por mês. Identifica o vendedor destaque e os produtos abaixo da média.

### 02 — Pipeline de Qualidade de Dados
Valida um cadastro de clientes verificando nome, e-mail, CPF e idade. Separa os registros válidos dos inválidos e gera um relatório detalhando os erros encontrados em cada registro.

### 03 — Monitoramento de Estoque
Analisa o estoque de uma farmácia identificando produtos zerados, abaixo do mínimo e vencidos. Gera alertas por produto e calcula o valor total em estoque por categoria.

## Conceitos aplicados
- Dicionários como acumuladores de dados
- Loops com extração de múltiplas dimensões
- Validação de dados com múltiplos critérios
- Fatiamento de strings para extração de datas
- Funções max() e min() com key=
