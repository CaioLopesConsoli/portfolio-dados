# Projetos SQL — Engenharia de Dados

Consultas SQL desenvolvidas para análise de dados em um banco de dados real de e-commerce (SQLite).

## Projetos

### 01 — Análise de Comportamento de Clientes
Identifica os clientes mais valiosos, calcula ticket médio por cliente, encontra clientes inativos e analisa padrões de compra.

### 02 — Performance de Produtos
Analisa os produtos mais vendidos, faturamento por categoria, produtos sem vendas e métricas gerais de itens por pedido.

### 03 — Análise Temporal de Pedidos
Examina a distribuição de pedidos por mês, identifica o mês de maior faturamento e analisa a distribuição percentual por status.

## Conceitos aplicados
- JOIN triplo entre múltiplas tabelas
- Subconsultas no FROM, SELECT e WHERE
- LEFT JOIN com IS NULL para dados ausentes
- GROUP BY com funções de agregação
- Cálculo de percentuais com subconsulta
- Extração de datas com SUBSTR
